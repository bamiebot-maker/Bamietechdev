BamietechDev Portfolio Website
A modern, responsive personal portfolio website for Ibrahim Sobur Bamidele (Bamietech Dev) showcasing skills, projects, and achievements.

🚀 Features
Modern Design: Clean, professional UI with orange color theme

Fully Responsive: Works perfectly on desktop, tablet, and mobile devices

Interactive Elements: Smooth animations, hover effects, and scroll transitions

Contact Form: Integrated with Formspree for email submissions

Project Showcase: Beautiful project cards with image placeholders

Download Resume: Direct download functionality for resume/CV

📋 Sections
Hero: Introduction with call-to-action buttons

About: Personal bio and statistics

Skills: Technical skills with icons

Projects: Project portfolio with descriptions

Achievements: Leadership roles and accomplishments

Education: Academic background and training

Contact: Contact information and form

🛠️ Technologies Used
HTML5: Semantic markup

CSS3: Modern styling with Flexbox/Grid

JavaScript: Interactive functionality

Font Awesome: Icons

Google Fonts: Inter font family

Formspree: Contact form handling

⚙️ Setup Instructions
Prerequisites
A modern web browser

Basic knowledge of HTML/CSS/JavaScript (for customization)

Local Development
Clone or download the project files

Create the folder structure as shown above

Add your images to the img/ folder:

img/bamie.jpg - Your profile photo

img/projects/ - Project screenshots

Add your resume as assets/Ibrahim_Sobur_Bamidele_Resume.pdf

Open index.html in your browser to view the website locally

Formspree Setup (Contact Form)
Go to Formspree.io

Create a free account

Create a new form and get your form ID

Replace the Formspree URL in script.js:

javascript
// In the form submission section, change:
'https://formspree.io/f/xyzbblen'
// to your actual Formspree URL
🚀 Deployment
Option 1: Netlify (Recommended)
Push your code to a GitHub repository

Go to Netlify

Drag and drop your project folder or connect your GitHub repo

Your site will be deployed instantly

Option 2: Vercel
Push your code to a GitHub repository

Go to Vercel

Import your GitHub repository

Deploy with default settings

Option 3: GitHub Pages
Create a repository named username.github.io

Upload all files to the repository

Go to Settings → Pages → Select main branch

Your site will be available at https://username.github.io

🎨 Customization
Colors
The primary color theme is orange. To change colors, modify the CSS variables in style.css:

css
:root {
    --primary-color: #FF6B35;
    --primary-dark: #E55A2B;
    --primary-light: #FF8E53;
    /* ... other variables */
}
Content
Update the following in index.html:

Personal information

Project descriptions

Skills list

Achievement details

Contact information

Images
Replace placeholder images in the img/ folder:

Profile photo: img/bamie.jpg

Project images: img/projects/ folder
